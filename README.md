# NexaChat 🤖

AI-powered Angular developer assistant built with Angular 21 Signals, 
Groq LLM API, and Node.js streaming proxy.

## Features
- Real-time streaming responses
- Conversation history with localStorage
- Sidebar with 10 conversation limit
- Markdown rendering for code responses
- Secure server-side API key management
- Angular 21 Signals — no NgRx

## Tech Stack
Angular 21 · TypeScript · Tailwind CSS · Groq API · Node.js · Express

## Run locally
# Terminal 1
ng serve

# Terminal 2  
node server/proxy.js
